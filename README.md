# Final-Lab-Project
Manahil Fatima (FA25-BSE-060) &lt;br> Mishal Shahid(FA25-BSE-169)
This project is a JavaFX application called Secret Message Encoder/Decoder that provides secure text encryption and decryption. It allows users to protect messages using two different modes: Secret Characters (vowel substitution and shifting) and Morse Code.

Key Features:

Password Protection: Users must create a password during encryption, which is required to unlock and decrypt the message later.

Modular Design: The system uses interfaces like EncryptionLogic to easily switch between different encryption methods.

User-Friendly Interface: A guided workflow takes users through welcome, selection, input, and result screens.

OOP Implementation: The code demonstrates core programming concepts like Polymorphism, Encapsulation, and Inheritance.
